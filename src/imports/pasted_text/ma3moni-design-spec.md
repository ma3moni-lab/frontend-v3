Design a **highly professional, modern, and production-ready UI/UX system** for a compatibility-first, marriage-oriented relationship platform called **Ma3moni**.

The platform is built for serious individuals seeking meaningful, long-term relationships. The design must prioritize **clarity, simplicity, trust, and seamless interaction flow**, while also supporting **robust administrative control, moderation, and operational efficiency**.

---

# CORE DESIGN PRINCIPLES

* Simplicity over complexity
* Clear visual hierarchy and spacing
* Seamless, frictionless user journeys
* Calm, trustworthy, and premium interface
* Intentional interaction (no addictive or chaotic patterns)
* Scalable and consistent design system

---

# DESIGN STYLE DIRECTION

* Style: Minimal, premium, modern
* Inspiration: Stripe, Airbnb, Apple-level clarity
* Typography: Clean, highly readable, elegant
* Color Palette:

  * Neutral base (white, soft gray)
  * Primary: Deep blue or teal (trust and stability)
  * Accent: Subtle warm tone (used sparingly)
* Generous whitespace
* No visual clutter or unnecessary decoration

---

# USER EXPERIENCE (MOBILE-FIRST)

## 1. Onboarding Flow

* Step-by-step guided onboarding
* Clear progress indicator
* Short, digestible screens
* Inputs:

  * Personal details
  * Values, lifestyle, personality
  * Communication style
  * Life goals
  * Marriage timeline
* Partner preferences with weighting:

  * Must-have / Important / Nice-to-have
* Smooth transitions and micro-interactions

## 2. User Dashboard

* Profile completion percentage indicator
* Sections:

  * Profile summary
  * Compatibility preferences
  * Matches overview
  * Messages overview
  * Activity report
* Subscription status display
* Clean layout with collapsible sections
* Photo upload (maximum of 5 images)

---

## 3. Compatibility & Matching

* Match cards showing:

  * Compatibility score

* Detailed compatibility breakdown:

  * Visual bars or simple charts

* “Top Matches” prioritization

---

## 4. Messaging System

* One-message initiation system
* Request/accept interaction flow
* Clean, distraction-free chat interface
* Optional guided first message suggestions
* Easy exit/report options

---

## 5. Subscription Flow

* Simple pricing UI
* Clear flow:

  * Select plan → Payment → Confirmation
* Transparent feedback states
* Payment integration-ready

---

## 6. Referral System

* Unique user ID for referrals
* Simple referral dashboard:

## 7. Notifications & Trust

# ADMIN DASHBOARD (WEB-BASED, SaaS-LEVEL)

Design a **clean, structured, and highly functional admin system** with full control over platform operations.

---

## ADMIN CORE FEATURES

### 1. Dashboard Overview

* Key metrics:

  * Total users
  * Active users
  * Matches
  * Revenue
* Clean charts and summaries

---

### 2. User Management System

* Full user list (table format)
* Ability to:

  * View full user profiles
  * Suspend users (with mandatory reason input)
  * Reactivate users
  * Delete users

---

### 3. Advanced Blocking / Blacklist System

Admin should be able to block/blacklist users using:

* Name
* Phone number
* Email address
* Profile picture

System should:

* Detect and prevent re-registration
* Flag suspicious duplicates

---

### 4. Profile Moderation System

* Approve or reject uploaded profile pictures
* Queue-based moderation interface
* Clear approve/reject actions
* Optional reason for rejection
* Status tracking (pending/approved/rejected)

---

### 5. Customer Care System

#### Customer Care Dashboard

* Dedicated interface for support agents
* Ticket-based or chat-based system
* View and respond to user issues

#### Features:

* Conversation history
* User profile access (limited)
* Escalation to admin
* Internal notes

---

### 6. Role & Permission Management

* Role-based access control (RBAC)
* Roles:

  * Super Admin
  * Admin
  * Customer Care Agent
* Permissions control:

  * What each role can view/edit
* Clean permission settings UI

---

### 7. Payments & Subscription Management

* Payment integration interface:

  * Input API keys (Stripe, Paystack, etc.)
* View transactions
* Manage subscriptions
* Refund/issue handling interface

---

### 8. Referral System Management

* Configure referral bonus amount
* Track referrals globally
* Monitor abuse or fraud

---

### 9. Moderation & Reports

* View reported users/content
* Action options:

  * Warn
  * Suspend
  * Ban
* Activity log for all moderation actions

---

### 10. Data Insights & Analytics

* User behavior insights
* Match success trends
* Gender balance tracking
* Subscription analytics
* Clean, simple data visualization

---

### 11. System Controls

* Platform settings
* Gender balance mechanisms
* Feature toggles

---

Generate a **complete, production-ready UI/UX system** including:
